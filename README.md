# Arduino-menjalnik-za-simulator-voznje
Modeli za 3D tiskalnik in Arduino koda, s katero lahko Arduino Leonardo spremenite v menjalnik s 6 prestavami + R za simulator vožnje!

POMEMBNO: Uporabite lahko le Arduino Leonardo, Micro ali druge mikrokontrolerje, ki so na tem seznamu: https://github.com/MHeironimus/ArduinoJoystickLibrary/wiki/Supported-Boards. Arduino Uno NE dela!


Poleg arduina in 3D tiskalnika potrebujete tudi:
- 2x potenciometer 10kOhm (https://www.amazon.de/dp/B00H969CM0?ref_=pe_111929571_1111661321_t_fed_asin_title&th=1)
- 1x tipka za mehansko tipkovnico 
- 1x navojna palica M6 dolžine 8,5cm
- 2x elastika
- nekaj izoliranih tankih žic za povezave
