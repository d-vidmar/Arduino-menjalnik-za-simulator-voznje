# Arduino-menjalnik-za-simulator-voznje
Modeli za 3D tiskalnik in Arduino koda, s katero lahko Arduino Leonardo spremenite v menjalnik za simulator vožnje s 6 prestavami + R in dodatnim gumbom!

POMEMBNO: Uporabite lahko le Arduino Leonardo, Micro ali druge mikrokontrolerje, ki so na [TEM SEZNAMU](https://github.com/MHeironimus/ArduinoJoystickLibrary/wiki/Supported-Boards). Arduino Uno NE dela!
Potrebujete tudi [TO KNJIŽNJICO](https://github.com/MHeironimus/ArduinoJoystickLibrary/archive/master.zip). <br/><br/>

## Seznam stvari, ki jih potrebujete (poleg 3D tiskalnika in spajkalnika)
- 1x Arduino Leonardo / Pro Micro / drugi mikrokontroler z zgornjega seznama
- 2x potenciometer 10kOhm ([amazon](https://www.amazon.de/dp/B00H969CM0?ref_=pe_111929571_1111661321_t_fed_asin_title&th=1))
- 1x tipka za mehansko tipkovnico 
- 1x navojna palica M6 dolžine 8,5cm
- 2x elastika
- nekaj izoliranih tankih žic za povezave <br/><br/>
